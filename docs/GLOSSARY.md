# Glossary — Agentic AI Concepts

- **Agent** — A system capable of reasoning, planning, acting, and reflecting.  

- **Planner** — Decomposes a goal into structured steps.  

- **Worker Agent** — Executes specific parts of a task.  

- **Tool** — External system an agent can interact with.  

- **ReAct** — Reason → Act → Observe → Reflect workflow.  

- **ToT (Tree-of-Thoughts)** — Branching reasoning.  

- **Memory** — Long-term, short-term, or episodic agent knowledge.  

- **Vector DB** — Storage for semantic embeddings.  

- **Reflection** — Model critiques its own reasoning.  

- **HITL** — Human-in-the-loop.  

- **MCP** — Model Context Protocol.  

- **A2A** — Agent-to-agent messaging protocol.  

- **SLIM / ACP** — Lightweight tool calling standards.  

- **Retriever** — Retrieves info from memory.  

- **Verifier** — Checks safety, correctness, and compliance.
